# Git & GitHub Practice Project

## Overview
This repository is created as part of the Rhombix Technologies DevOps Internship.
The purpose of this project is to demonstrate real-world Git and GitHub workflows.

## What This Project Covers
- Repository initialization
- Branch creation and management
- Committing changes with meaningful messages
- Merging feature branches into the main branch
- Pushing code to GitHub

## Tools & Technologies
- Git
- GitHub
- Visual Studio Code
- Linux (Ubuntu)

## Workflow Summary
1. Initialize repository
2. Create feature branch
3. Commit changes
4. Merge into main branch
5. Push updates to GitHub
